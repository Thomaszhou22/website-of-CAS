# 🏛️ 社团签到系统

一个功能完善的社团签到管理系统，支持社长、社员和管理员三种角色，提供签到码管理、活动时长统计、数据导出等功能。

## ✨ 主要功能

### 👨‍💼 社长功能
- 创建社团，生成唯一社团号
- 生成和管理签到码（6位字母数字混合）
- 设置活动时长（C/A/S三类）
- 审核社员签到
- 查看所有社员统计信息
- 管理活动日期

### 👥 社员功能
- 通过社团号进入社团
- 输入签到码进行签到
- 查看个人签到统计
- 实时查看活动时长（C/A/S）

### 🧑‍💼 管理员功能
- 查看所有社团统计
- 管理所有社团信息
- 查看详情和删除社团
- 账号密码管理

## 🚀 快速开始

### 本地部署
1. 下载所有文件到本地
2. 用浏览器打开 `index.html` 即可使用

### 在线部署
#### GitHub Pages（免费）
1. 创建 GitHub 仓库
2. 上传所有文件
3. 在仓库设置中启用 GitHub Pages
4. 访问：`https://你的用户名.github.io/仓库名`

#### Vercel（推荐）
1. 访问 [vercel.com](https://vercel.com)
2. 连接 GitHub 仓库
3. 一键部署

#### Netlify
1. 访问 [netlify.com](https://netlify.com)
2. 拖拽文件夹上传
3. 自动生成链接

## 📱 使用方法

### 首次使用
1. **注册管理员**：选择"管理后台"，点击"注册管理员"
2. **创建社团**：社长登录后，点击"创建社团"

### 日常使用流程
1. 社长生成签到码
2. 社长设置活动时长
3. 社员输入签到码签到
4. 社长审核签到
5. 查看统计数据

## 📊 数据说明

- 所有数据存储在浏览器本地（localStorage）
- 清除浏览器数据会丢失所有信息
- **重要数据请定期导出备份**
